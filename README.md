## ⚙️ Installation & Setup

1. Clone the Repository

```bash
git clone https://github.com/your-username/defi-option-vault.git
cd defi-option-vault
```

2. Install Dependencies

Frontend:

```bash
cd frontend
npm install
```

Smart Contracts:

```bash
cd contracts
npm install
```

AI Module:

```bash
cd ai-engine
pip install -r requirements.txt
```

3. Run Local Blockchain

```bash
npx hardhat node
```
